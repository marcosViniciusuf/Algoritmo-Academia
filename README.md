Este projeto consiste em um sistema de gerenciamento de exercícios físicos desenvolvido em C++, com armazenamento de dados em arquivo CSV para garantir persistência e organização das informações. A aplicação permite cadastrar, editar, remover e buscar exercícios por nome, grupo muscular ou carga, além de listar registros completos e realizar ordenações eficientes. O sistema utiliza estruturas dinâmicas, algoritmos de ordenação e busca binária, proporcionando melhor desempenho no tratamento dos dados. Desenvolvido com fins acadêmicos por alunos do primeiro período de Ciência da Computação, o projeto tem como objetivo aplicar conceitos fundamentais de programação, como manipulação de arquivos, estruturas de dados e organização lógica do código.
